## 📝 About the Project

**TTSdownloads** is a command-line interface (CLI) utility designed to download text-to-speech audio files utilizing the Google Translate API. It supports any language natively available on the platform.

## ⚡ Getting Started

### Prerequisites
Make sure you have Python installed on your system.

### Installation & Usage
```bash
# 1. Clone the repository
git clone [https://github.com/DevPolar003/TTSdownloads.git](https://github.com/DevPolar003/TTSdownloads.git)

# 2. Navigate to the project directory
cd TTSdownloads

# 3. Run the application
python main.py

.
├── frases.txt    # Input file containing phrases to process
└── main.py       # Main application entry point
