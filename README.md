# TTSdownloads

> A Python CLI tool to download text-to-speech audio for language flashcards.

## 📝 Sobre o projeto

TTSdownloads é um CLI com o proposíto de baixar audios usando a API do google translator, para qualquer língua disponível.

## ⚡ Como rodar

```bash

# 1. Clone o reposítorio
git clone https://github.com/DevPolar003/TTSdownloads.git

```
